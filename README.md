# To-Do List App MVP

This project is now a small Flask web app with a more polished visual design.

## Features

- Add tasks
- View all saved tasks
- Mark tasks complete
- Persist tasks in a local `tasks.json` file
- Show lightweight progress stats in the UI

## Setup

Install Flask:

```bash
pip install flask
```

## Run the app

```bash
python todo_app.py
```

Then open the local URL shown in the terminal, usually:

```text
http://127.0.0.1:5000
```

## Project structure

- `todo_app.py` contains the Flask routes and task storage logic
- `templates/index.html` contains the page structure
- `static/css/style.css` contains the custom visual design
- `static/js/app.js` adds small UI enhancements

## MVP flow

1. Add a task like `Finish assignment`
2. View it instantly in the task board
3. Mark it done when complete

This keeps the original MVP scope while making the experience more appealing.
